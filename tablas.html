Etiquetas usadas

table: la etiqueta que encierra una tabla

tr(table row) contruye una fila

td (table data) construye una celda

caption:para el titulo de la tabla

thead para la cabecera

th para cabeceras por fila

rowspan para que una celda ocupe varias filas

colspan para que una celda ocupe mas de una columna
